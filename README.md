# nodevid19

Se trata de una solución desarrollada en NodeJS que consume datos sobre el corona virus (COVID-19) por pais.

### Instalación
```bash
npm install
```

### Ejecución
**help**
```bash
node index.js --help
```

**consulta**
```bash
node index.js -p argentina

o

node index.js --pais argentina
```
